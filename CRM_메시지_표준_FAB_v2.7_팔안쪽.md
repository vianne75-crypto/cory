# cory CRM 자동화 메시지 표준

> SAGE 전사 공지 2건 반영 — **FAB v2.7**(2026-06-04 대표 정제) + **"팔 안쪽" 부착부위 표준**(2026-06-04).
> cory에서 발신하는 모든 CRM 이메일·SMS·카카오 알림톡·재구매 안내·이탈 윈백 메시지는 본 표준을 따른다.

## 1. 점검 결과 (2026-06-09 전수 스캔)

| 항목 | 결과 |
|------|------|
| 금지 부착부위 표현(`손등`·`팔뚝`·`상박`) — 코드·GAS·HTML | **0건** ✅ |
| 구버전 FAB 정량 클레임(`2배`·`효소결핍`·`참여율` 등) — 발신 템플릿 | **0건** ✅ |
| webhook 회신 텍스트(cloudflare-worker·hc-sync-gas) | 사용자 대면 카피 없음(JSON 응답) ✅ |

→ **정정 대상 없음.** 본 문서는 향후 신규/개정 메시지의 준수 기준으로 적용.

발신 템플릿 현황:
- `gas/f1-launch-mailer.gs` — F1 키트 출시 CRM 이메일 (고객 대면) → **개정 시 §2 FAB 적용**
- `gas/winback-alert.gs` · `gas/order-alert.gs` — **내부 알림**(운영자 수신), FAB 불필요
- `gas/hc-followup-sheet.js` — 내부 팔로업 시트

## 2. FAB v2.7 표준 (전사 표준)

- **F**: 음주·흡연에 더 취약한 ALDH2 효소결핍을 확인하는 체감형 금연·절주 교육 패치
- **A**: 단 7분 체험으로 캠페인 참여율·인지도 최대 2.5배 *(KHEPI·보건복지부 우수사례집 2023~2026, 9개 보건소·1개 대학단체 누적 기준)*
- **B**: 막연했던 위험이 피부로 와닿는 *내 이야기*가 됩니다. 참여자가 저절로 모여드는 캠페인 우수사례로 남습니다.

**B2G 메시지 4원칙**: ①정량(A)/정성(B) 분담 ②기관 단위 호명 ③자명 정보 생략 ④학술키워드+직관단서 동시.

### 단축 채널형 (카카오 알림톡·SMS — 글자수 제약)
