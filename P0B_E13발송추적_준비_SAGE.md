## 3. 조건부 서식 (입력 후 설정)

- **G열 repurchase_yn = "Y"** → 행 전체 배경 연초록 (#e8f5e9)
- **G열 repurchase_yn = "N"** → 행 전체 배경 연분홍 (#fce4ec)
- **E열 발송일 기준 +180일 경과 + G열 "대기"** → 배경 노랑 (#fff9c4) — 6개월 추적 만료 경고

## 4. 파일럿 발송 대상 기관 (E13 출시 전 사전 정의)

> 기준: cory purchase_stage = **만족·추천** (재구매 확정 기관) 중 교육자료 니즈 확인된 곳
> → E13 출시 즉시 1순위 발송 대상

### Tier 1 — 즉시 발송 (보건소·전문기관 재구매 확인)

| cory_id | 기관명 | 세그먼트 | 근거 |
|--------|------|---------|------|
| (경주시보건소) | 경주시보건소 | 보건소 | B4 VOC 실제 사용 기관 |
| (울주군보건소) | 울주군보건소 | 보건소 | B4 VOC 실제 사용 기관 |
| (창원중독관리통합지원센터) | 창원중독관리통합지원센터 | 전문센터 | B4 VOC 실제 사용 기관 |
| (강남구보건소) | 강남구보건소 | 보건소 | FORGE 현장VOC 기관 |
| (부산중독관리센터) | 부산중독관리센터 | 전문센터 | FORGE VOC 에스컬레이션 기관 |

> ⚠️ cory_id는 대표 또는 PULSE가 cory 대시보드에서 직접 확인 후 기입 필요.

### Tier 2 — E13 출시 후 2주 내 발송

- purchase_stage = 만족·추천 전체 45개 기관
- 세그먼트 기준: 전문센터 > 보건소 > 학교 > 사업장 순

## 5. 운영 프로토콜

**E13 발송 시 즉시:**
- SAGE가 E13 발송 시 시트에 행 추가 (A~F열 입력)
- G열 = "대기" 로 시작

**매주 월요일 (P0-E 주간 루틴 연계):**
- cory 대시보드에서 재주문 기관 확인
- 해당 기관 G열 "Y" + H열 재구매일 입력
- I열 days_to_repurchase 자동 계산 확인

**6개월 경과 "대기" 기관:**
- 노랑 경고 표시 → SAGE가 cory purchase_stage 재확인 후 "N" 처리 또는 팔로업 재시도
