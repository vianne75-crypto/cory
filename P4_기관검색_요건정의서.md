# P4 기관 검색 기능 요건 정의서

> 발신: PULSE 다온 | 수신: FLUX 이음 (X4 구현용)

---

## 현황 (이미 구현된 것)

`admin-institutions.js` `matchesSearch()` 함수가 아래 필드에서 클라이언트 사이드 검색 지원:
- 기관명 (`name`)
- 시군구 (`district`)
- 담당자명 (`metadata.contact_name`)
- 연락처 (`metadata.contact_phone`)
- UTM코드 (`metadata.utm_code`)

필터: 유형 / 지역 / 구매단계 / 교육도입 수준 ← **이 부분은 추가 작업 불필요**

---

## GAP — 추가 필요 기능

### G1. 주문번호(order_idx) → 기관 역추적

**사용 케이스:** 대표가 wcolive 주문 목록을 보다가 주문번호 입력 → 어느 기관인지 즉시 확인

```
입력: "137555"  →  결과: 춘천시보건소 카드로 이동
입력: "137535"  →  결과: 성동구고혈압당뇨병등록교육센터
```

**구현 방식 (FLUX 이음에게):**

검색어가 숫자 6자리이면 order_idx 검색으로 자동 분기:

```javascript
// matchesSearch() 내 추가 로직 (클라이언트 사이드)
// instCache 로드 시 orders도 함께 로드하거나,
// 숫자 6자리 입력 시 Supabase 쿼리로 institution_id 조회

async function findInstitutionByOrder(orderIdx) {
  const { data } = await supabase
    .from('orders')
    .select('institution_id')
    .eq('order_idx', orderIdx)
    .eq('matched', true)
    .limit(1);
  return data?.[0]?.institution_id || null;
}
```

검색창에 6자리 숫자 입력 → 매칭 기관 1건으로 결과 필터링 → 자동 상세 패널 열기

---

### G2. 상담 태그 / 키워드 → 기관 목록

**사용 케이스:** "일터학회" 입력 → 해당 태그가 붙은 상담내역이 있는 기관 목록

**구현 방식:**

검색어 앞에 `#` 붙이면 태그 검색으로 분기:

```
입력: "#일터학회"  →  consultations 테이블에서 tags 배열에 해당 태그 포함 기관 목록
입력: "#HC팔로업"  →  HC 팔로업 기록이 있는 기관 전체
```

```javascript
async function findInstitutionsByTag(tag) {
  const { data } = await supabase
    .from('consultations')
    .select('institution_id')
    .contains('tags', [tag])
    .not('institution_id', 'is', null);
  return [...new Set(data.map(r => r.institution_id))];
}
```

결과: instFiltered를 해당 institution_id 목록으로 교체
