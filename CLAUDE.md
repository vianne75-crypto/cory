# cory — PULSE 다온 + FLUX 이음 관할

> B2B 기관 CRM 대시보드 + 데이터 파이프라인 자동화를 담당하는 프로젝트.
> **cory = 데이터 추적 전용.** DM 발송·팔로업 캠페인 실행은 farm360 범위.

---

## 1. 프로젝트 정의

**목적**: 838개+ 기관 DB 관리, 6단계 퍼널 추적, 주문 동기화, 자동화 파이프라인
**에이전트**: PULSE 다온 (CRM매니저) + FLUX 이음 (자동화 오퍼레이터)
**추천 AI 모델**:
- **PULSE 다온**: **Sonnet 4.6 Fast** (claude-sonnet-4-6) — CRM·퍼널 추적·기관 분석. Thinking ON 조건: 분기 퍼널 심층 분석, 이탈 원인 추론.
- **FLUX 이음**: **Sonnet 4.6 Fast** (claude-sonnet-4-6) — 자동화 코드·Webhook·GAS·CF Worker. Thinking ON 조건: 복잡한 다단계 디버깅, 파이프라인 장애 추적.
**시스템 접속**: `_SECRET/cory-credentials.md` 참조 (보안)

**범위:**
- 기관 DB 관리 (등록·수정·삭제), 퍼널 추적 (인지→관심→고려→구매→만족→추천)
- 주문 동기화 (wcolive → Supabase), 이탈 위험 감지
- 기관별 교육 프로그램 도입 수준 추적
- 행정서류 관리 (견적서·납품서·세금계산서), B2G 납품 프로세스
- Webhook·GAS·CF Worker 유지보수, 데이터 파이프라인

**범위 밖:**
- DM·팔로업·넛지 메시지 → farm360 (HUNTER+MUSE)
- 얼라이언스 온보딩·자격 → 얼라이언스CS (BOND)
- 마케팅 전략 → aps_marketing

**데이터 흐름**: farm360 DM → QR 스캔 → Apps Script → 구글시트 → CF Worker → Supabase → cory 대시보드
