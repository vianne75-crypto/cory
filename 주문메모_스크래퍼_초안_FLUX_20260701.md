# 주문 관리자 메모 스크래퍼

> wcolive 관리자 주문목록의 **관리자 메모칸**(입완·전세발행 등)을 긁어 cory `orders.memo_raw`에 기록.
> 상담 스크래퍼(`sangdam_list.htm`)와 동일 방식 — 대표 로그인 세션에서 콘솔 실행.
> ✅ **구조 확정 (2026-07-01 대표 테스트)**: 목록=`jumun_list.htm` 표2, **고유번호=order_idx / 메모=관리자메모**. 총 주문 1,461건.

## 데이터 흐름
```
wcolive 관리자 로그인 → 주문목록 페이지 → [콘솔 스크립트 실행]
  → 각 행에서 {주문번호, 관리자메모} 추출
  → POST https://aps-webhook.vianne75.workers.dev/sync-order-memos
  → orders.memo_raw 갱신 (order_idx 기준) + memo_parsed=false
→ cory 관리자에서 "메모 파싱" 버튼 클릭
  → admin-memo-parser 실행 → payment_confirmed·invoice_issued 자동 반영
```

## 확정된 구조 (2026-07-01)

| 항목 | 값 |
|------|-----|
| 주문목록 URL | `/admin/sub_sale/jumun_list.htm?ajax_yn=0` (page= 파라미터) |
| 데이터 테이블 | 헤더에 "고유번호"+"메모" 포함된 표 (표2) |
| order_idx 컬럼 | **고유번호** (셀1, 예: 138655) |
| 메모 컬럼 | **메모** (셀7 — 관리자 메모, 입완/전세) |
| 페이지당 | ~15건 / 총 1,461건 (~98p) |

> 요구사항(셀3)·전달메세지(셀4)는 고객 요청이므로 제외. 관리자 메모는 셀7 "메모"만.

## 스크래퍼 (헤더 자동 탐지형 — 구조 확정 반영)

```javascript
(async function(){
  // ── 설정 ──
  const ORDER_LIST_URL = '/admin/sub_sale/jumun_list.htm?ajax_yn=0'; // 확정
  const PAGES = 10;             // 긁을 페이지 수 (최근분 우선, 필요시 늘림)
  const W = 'https://aps-webhook.vianne75.workers.dev/sync-order-memos';
  const ORDER_HEADERS = ['고유번호','주문번호','주문no','ordno']; // 고유번호 확정
  const MEMO_HEADERS  = ['메모','관리자메모','주문메모','admin_memo']; // 메모 확정

  const findCol = (headerCells, cands) => {
    for (let i=0;i<headerCells.length;i++){
      const t=(headerCells[i].textContent||'').replace(/\s/g,'').toLowerCase();
      if (cands.some(c=>t.includes(c.replace(/\s/g,'').toLowerCase()))) return i;
    }
    return -1;
  };

  const all=[]; let err=0;
  console.log('=== 주문메모 스크래핑 시작 ('+PAGES+'p) ===');
  for(let p=1;p<=PAGES;p++){
    try{
      const r=await fetch(ORDER_LIST_URL+(ORDER_LIST_URL.includes('?')?'&':'?')+'page='+p,{credentials:'same-origin'});
      const h=await r.text();
      const doc=new DOMParser().parseFromString(h,'text/html');
      // 데이터 테이블 추정: 헤더에 주문번호 후보가 있는 첫 table
      const tables=[...doc.querySelectorAll('table')];
      let table=null, oc=-1, mc=-1;
      for(const t of tables){
        const head=t.querySelector('tr');
        if(!head)continue;
        const hc=head.querySelectorAll('th,td');
        const o=findCol(hc,ORDER_HEADERS), m=findCol(hc,MEMO_HEADERS);
        if(o>=0 && m>=0){ table=t; oc=o; mc=m; break; }
      }
      if(!table){ console.warn('p'+p+': 주문번호+메모 컬럼 테이블 못찾음'); continue; }
      const rows=table.querySelectorAll('tr');
      for(let i=1;i<rows.length;i++){
        const c=rows[i].querySelectorAll('td,th');
        if(c.length<=Math.max(oc,mc))continue;
        const order_idx=(c[oc].textContent||'').trim().replace(/[^0-9]/g,'');
        const memo=(c[mc].innerText||c[mc].textContent||'').trim();
        if(order_idx && memo) all.push({order_idx, memo});
      }
      console.log(p+'/'+PAGES+' 누적 '+all.length+'건');
      await new Promise(z=>setTimeout(z,100));
    }catch(e){console.error('p'+p+':'+e.message);err++;if(err>10)break;await new Promise(z=>setTimeout(z,500));}
  }
  console.log('스크래핑 완료: '+all.length+'건 (메모 있는 주문만)');
  if(!all.length){console.warn('메모 추출 0건 — URL/컬럼 헤더 확인 필요');return;}

  // 전송 (200건 배치)
  let up=0;
  for(let i=0;i<all.length;i+=200){
    const b=all.slice(i,i+200);
    try{
      const r=await fetch(W,{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify(b)});
      const j=await r.json();
      if(j.success){up+=j.updated||0;console.log('배치'+(Math.floor(i/200)+1)+': updated '+j.updated+', skipped '+j.skipped);}
      else console.error(j.error);
    }catch(e){console.error(e.message);}
  }
  console.log('=== 완료: memo_raw '+up+'건 갱신 → 이제 cory 관리자에서 [메모 파싱] 클릭 ===');
})();
```
