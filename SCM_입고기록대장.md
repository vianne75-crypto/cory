## 이번 입고 강화 검수 방침 (2026-06-08분)

### 샘플링 (60,100개 / 6박스 전수 불가 → 박스별 강화 샘플링)
1. **박스 6개 전부**에서 각 **무작위 100개** 추출 → 총 600개 검사
2. 어느 박스든 critical 불량 **3개 이상** 발견 시 → 해당 박스 **전수검사**로 격상
3. 전체 샘플 불량률 **2% 초과** 시 → 제조사 클레임 + SAGE 즉시 보고
