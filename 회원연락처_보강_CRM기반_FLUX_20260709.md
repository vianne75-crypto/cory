# 회원 연락처 자동 보강 (CRM 자동화 기반)

> **문제**: 애니빌드 주문 webhook이 **이메일·전화를 안 실어옴** → 주문마다 담당자 연락처 누락 → 세금계산서·CS·CRM 자동화 불가. (구매자명 j_name/s_name은 있음, 연락처만 없음)
> **원인**: 이메일·전화는 **회원(mem_id) 정보엔 있으나** 주문 페이로드에 미포함.
> **해결**: 회원 연락처를 `member_contacts`에 수집 → `order.mem_id` 조인으로 **모든 주문에 연락처 자동**. 대리점 사업자번호 자동화(dealer_business_info)와 동일 구조.

## 구조
```
[신규·수정 회원]  애니빌드 회원 webhook(/anybuild-member) → member_contacts (전 회원 자동)
[기존 회원]       wcolive mem_list 스크래퍼 1회 → /sync-member-contacts → member_contacts
[활용]           주문·세금계산서·CS = order.mem_id ⟕ member_contacts.email/phone (자동)
```

## 구축 (완료분 / 대표 액션)
- ✅ `member_contacts` 테이블 스키마 `data/member-contacts-schema.sql` — **대표 Supabase 실행**
- ✅ worker `/anybuild-member` 확장 — 전 회원 email·phone → member_contacts (대리점 아니어도) + `/sync-member-contacts` 백필 엔드포인트 — **대표 aps-webhook 재배포**
- ⬜ 백필 스크래퍼 1회 실행 — 아래

## 백필 스크래퍼 (전 회원 연락처 — 대표 wcolive 콘솔 1회)

```javascript
(async function(){
  const U='/admin/sub_member/mem_list.htm?ajax_yn=0', MAXP=300;
  const W='https://aps-webhook.vianne75.workers.dev/sync-member-contacts';
  const all=[]; let er=0, empty=0;
  console.log('=== 전 회원 연락처 추출 시작 ===');
  for(let p=1;p<=MAXP;p++){
    try{
      const r=await fetch(U+'&page='+p,{credentials:'same-origin'});
      const doc=new DOMParser().parseFromString(await r.text(),'text/html');
      const t=[...doc.querySelectorAll('table')].find(t=>{const hd=t.querySelector('tr');return hd&&/사업자번호/.test(hd.textContent||'');});
      if(!t){if(++empty>=3)break; continue;}
      const rows=t.querySelectorAll('tr');
      if(rows.length<=1){if(++empty>=3)break; continue;}
      empty=0;
      for(let i=1;i<rows.length;i++){
        const tr=rows[i], c=tr.querySelectorAll('td,th');
        const idm=(tr.innerHTML.match(/mem_login\('([^']+)'\)/)||[])[1]||'';
        if(!idm)continue;
        const g=k=>((c[k]||{}).innerText||(c[k]||{}).textContent||'').trim().replace(/\s+/g,' ');
        // 셀 매핑: 12 상호 · 13 회원명 · 17 이메일 · 18 핸드폰 · 20 회원등급
        all.push({mem_id:idm, company:g(12), name:g(13), email:g(17), phone:g(18), memlv:g(20)});
      }
      if(p%20===0)console.log(p+'p, 회원 '+all.length+'명');
    }catch(e){console.error('p'+p+':'+e.message);if(++er>10)break;await new Promise(z=>setTimeout(z,400));}
    await new Promise(z=>setTimeout(z,80));
  }
  console.log('=== '+all.length+'명 추출 (이메일 '+all.filter(x=>x.email).length+') ===');
  let up=0;
  for(let i=0;i<all.length;i+=200){
    const b=all.slice(i,i+200);
    try{
      const r=await fetch(W,{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify(b)});
      const j=await r.json();
      if(j.success){up+=j.upserted||0;console.log('배치'+(Math.floor(i/200)+1)+': upserted '+j.upserted);}else console.error(j.error);
    }catch(e){console.error(e.message);}
  }
  console.log('=== 완료: member_contacts '+up+'명 백필 ===');
})();
```

## 활용 (다음 단계)
- 세금계산서 발행: order.mem_id → member_contacts.email 자동 (수동조회 종료)
- CRM 대시보드: 주문·기관에 담당자 이메일·전화 표시
- 향후: order-worker 수신 시 member_contacts 조인해 order에 contact 스냅샷(선택)
