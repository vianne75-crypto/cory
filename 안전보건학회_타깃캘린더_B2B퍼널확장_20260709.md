## 🎯 실행 권고 (KISS 외 즉시 착수 3~5)

1. **대한민국 안전산업박람회(9월 부산)** — KISS 다음 규모 부스형. **부스 참가 신청 검토**(9월이라 준비 시간 有). 방문자 명부 확보로 A1식 B2B 리드 자산화.
2. **기산협 세미나(5월)** — **보건관리자가 곧 청중**(가장 순수한 타깃). 후원/부스/미니세미나. 200명 규모라 접근 용이·고밀도.
3. **대한금연학회·알코올건강행동학회** — APS **코어 주제(금연·음주)** 세미나 발표/후원 → 전문가 권위 + KHEPI·복지부·상담사 네트워크(B2G 접점).
4. **한국산업간호협회** — 산업간호사=보건관리자 직결. 후원·부스로 저비용 고정밀 도달.
5. **직업환경의학회 추계(10월 부산)** — 사업장 검진 전문의 대상, B2B 신뢰 구축.

**접근 방식 매트릭스**:
- 부스형(명부 확보) = KISS·안전산업박람회·건설안전박람회
- 세미나/후원(권위·네트워크) = 금연학회·알코올학회·직업환경의학회·산업보건학회
- 직접 타깃(보건관리자) = 기산협·산업간호협회

## ⚠️ 데이터 공백 (검증 필요 — 세션 리셋 후 or 공식사이트 확인)
- 🔶 표시: 산업간호협회·알코올건강행동학회·보건교육건강증진학회·KIHA의 **2026/2027 정확한 일정·부스 가능 여부** 미확정 → 공식 홈페이지 직접 확인 필요.
- 부스/후원 **비용·신청 마감** 각 행사별 확인 필요(예산 판단용).
- ⚠️ 서브에이전트 전수 검증은 세션 한도(11:30 KST 리셋)로 미완 → 리셋 후 Tier1·2 심층 검증(정확 일정·부스비·참가규모) 가능.
