## 2. 조기 마감 (재고 소진 시)

체험팩 실물이 떨어지면 즉시 마감:

**Supabase SQL Editor:**
```sql
UPDATE public.settings SET value = jsonb_set(value,'{closed}','true')
WHERE key='kiss_counter';
```
→ 랜딩이 자동으로 "마감" 화면으로 전환됨.

**다시 열기** (재고 보충 시):
```sql
UPDATE public.settings SET value = jsonb_set(value,'{closed}','false')
WHERE key='kiss_counter';
```

**정원 조정** (100 → 다른 수):
```sql
UPDATE public.settings SET value = jsonb_set(value,'{max}','120')
WHERE key='kiss_counter';
```
