# SCM 품질 이슈 — 유한플라스틱 브리스터 캡

> FLUX(이음) → SAGE 슬기 핸드오프. 부자재 공급사 품질 이슈 + 제조사 클레임 전략 판단 요청.

## 핸드오프 5요소

- **요약**: 브리스터 캡 공급사 **유한플라스틱**의 직전 입고분에서 **약 10% 불량(약 1만 개)** 발생. 대표가 발주처 대표에게 구두 통보 완료. 이에 이번 2026-06-08 입고분(홀쭉이=알쓰패치용, 103,300개)부터 **입고검수 강화** 적용 예정.

- **핵심 발견**:
  1. 불량률 **10% = SCM 매뉴얼 KPI(불량률 <2%)의 5배 초과** — 매뉴얼 §2-③ "2% 초과 시 즉시 SAGE 보고 + 제조사 클레임" 대상
  2. **단가 인상과 품질 저하 동시 발생** — 유한플라스틱 단가 33→38원 인상(2026-05-01 출고분부터, 중동 PET 폭등) 직후 입고에서 10% 불량. **가격은 올리고 품질은 떨어진** 구조 → 클레임 협상 레버리지
  3. 신창 의존 67%·유한플라스틱 단일 공급 구조 — 품질 리스크 분산 필요 (FORGE 원가구조 정합)
  4. Supabase `inventory_in`·`loss_records` 테이블 미생성 → 현재 불량 데이터는 `SCM_입고기록대장.md` 수기 기록

- **요청** (SAGE 전략 판단 3건):
  1. **제조사 클레임 전략** — 단가 인상분(개당 +5원 × 10만개 = 50만원/회) 대비 10% 불량 손실(약 1만개 × 38원 = 38만원/회) 협상 카드화. 클레임 공식화 여부·수위 판단
  2. **불량률 KPI 정식 트래킹 등록** — 입고 건별 불량률을 운영 대시보드 지표로 등록할지 (현재 수기)
  3. **공급선 이원화 검토 여부** — 단일 공급사 품질 리스크 대응

- **긴급도**: 🟡 금주 (이번 강화검수 결과 회수 후 클레임 시점 판단)

## ✅ 클레임 타결 (2026-06-30, 대표 직접 협상)

> 6/8 입고분 대금 입금 시점에 대표가 **"불량 차감 없이 청구"**한 점을 문제 제기 → 공급사 보상 합의.

**합의 내용:**
- **보상 = 다음 발주 시 15,000개 무상 추가제작** (직전 불량 약 1만개를 초과하는 보상)
- **다음 발주 규칙: 115,000개 발주 → 청구는 100,000개** (15,000개 무상)

**경제 효과:**
- 다음 발주 실질 단가 = 100,000 × 38원 ÷ 115,000개 = **약 33원/개** → 단가 인상분(33→38원) 사실상 상쇄
- 6/8 입고분(103,300개 × 38원 = 3,925,400원)은 청구액 그대로 입금, 보상은 차기 발주로 이연

**FLUX 적용 (다음 발주 시):**
- 발주서·입고기록대장에 "발주 115,000 / 청구 100,000 / 무상 15,000" 명시
- 입고 검수는 강화 기준 유지(박스별 샘플링), 불량률 재발 시 즉시 SAGE 보고
