## 핸드오프 (5요소)

- **요약**: BOND 1차검증(재현신호 5패턴)의 CRM 반영 요청 5건을 **전부 cory에 반영 완료**. 상담 태그·`next_followup_date`·신규기관·중복병합·트리거 태그 신설.
- **핵심 발견**: ⭐ BOND가 짚은 **"보관분 출고요청 = 재구매 최강 조기신호"**를 `보관분_출고요청` 태그로 신설·이벤트화 → 향후 자동 팔로업 트리거 기반 확보.
- **요청 (PULSE→BOND/SAGE/HUNTER)**:
  - HUNTER: 대학 HC `예산사이클_9-11월` 태그 6건 반영됨 → 9~11월 예산기 스크립트 착수 가능.
  - SAGE: 재구매 동인(보관분·절주행사·강사연계) ICP v3.1 피드는 BOND 라우팅 유지.
  - BOND: 덕양구 통화 **전문 하단 첨부**(truncation 해소).
- **긴급도**: 🟢 완료 (상시 루프로 전환)

## ② `보관분_출고요청` 트리거 태그 신설 완료 (재구매 최강신호)

- **적용**: 동대문(200)·울주(300)·마포(200)·송파(SPT 250) 4건.
- **의미**: "작년 보관분 우선 배송해 달라" = 재구매 사이클 진입의 **명확한 조기신호**(BOND ✅✅ 최강 재현). 여러 트랙(1A·1C·mega) 공통.
- **향후 자동화 제안(FLUX)**: 신규 에이닷/주문 동기화 시 본문에 "보관분·보관품·재발송" 감지 → `보관분_출고요청` 자동 태그 → **재구매 팔로업·홍보동의·반응온도 재수집 접점** 자동 트리거(마포 온보딩 갭 계획과 정합). *현재는 수동 태그, 자동화는 별도 발주 시.*

## ③ 덕양구보건소 통화 전문 (truncation 해소)

> BOND 사본이 "절…"에서 잘림 → cory 원문(상담 id 605) 전문 전달.

**통화**: 2026-06-25 대표 직접 현장 방문 (웜미팅·납품 동행). 반응온도 🟢긍정(웜).

**요지 (원문 4점)**:
1. **7월 초 절주 행사 준비 순조** — 절주온(KHEPI)에서 전문강사 요청→**배정 완료**. 절주온 강사 신청 시 (APS로) 연락 옴.
2. **배정 강사 알쓰패치 사용 가능** + 담당자 본인도 사용 숙련.
3. **사업장 영업**: 기업 의뢰 환영(우리가 먼저 제안도 함), 단 관내(고양 덕양) 기업 수 적음.
4. **보건관리자 다수 방문** → 자사 홈페이지 통한 기업 소개 OK → **"기업은 (성과)보고서 쓰기 좋다."**

**PULSE 관찰(BOND 검증 보강용)**:
- ③④가 BOND 미포착 신규 정보 — **덕양구가 "사업장 교차영업 채널" 역할 자청**(보건소→관내 기업 소개). [[project_adoption_map|도입기관 공개지도]]의 사업장↔보건기관 양면시장 매칭 가설과 정합.
- ④ "기업은 성과보고서 쓰기 좋다" = **울주 인사이트 A(성과보고 주력등재=재구매 동인)와 동형** → 1C(울주)·1A(덕양구) 양 트랙에서 **"성과보고 증빙" 재구매 동인 2건째 확정**.
- ①②의 **절주온(KHEPI) 강사 연계**가 재구매 락인 요소 — 강사 배정→APS 연락 구조가 반복 접점 생성.
